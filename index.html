نانوكور NanoCore

كيرنل، وinitrd مبني يدوياً، وbootloader — بلا توزيعة جاهزة وبلا اختصارات. كل بايت بالنظام له سبب. A kernel, a hand-assembled initrd, and a bootloader — no base distro, no shortcuts. Every byte on the system earned its place.

01 / الفلسفةPHILOSOPHY

ليش سويته؟Why build this?

مو مشروع لازم يستخدمه أحد. المشروع نفسه هو الهدف: تفهم كل طبقة بين الضغط على زر التشغيل وظهور أول حرف على الشاشة. This isn't meant to replace anything you already run. The build itself is the point — understanding every layer between pressing power and the first character on screen.

01

ما فيه شي مخفيNothing hidden

ما كو توزيعة أم تحته. الكيرنل، سكربت الإقلاع، الموديولات — كلها انتخبتها وربطتها بإيدي، بايت ببايت، عشان أعرف بالضبط شنو يشتغل ولّيش. There's no parent distro underneath. The kernel, the init script, the modules — I chose and wired every one by hand, so I know exactly what runs and why.

02

الصغر مب نقصSmall is not missing

النسخة الأصغر توزن أقل من ٢.٣ ميجا وتقلعك على تيرمنال شغال خلال ثواني. كل ميجا زايدة لازم تدافع عن وجودها. The smallest build weighs under 2.3 MB and drops you into a working shell in seconds. Every extra megabyte has to earn its place.

03

BIOS، مو UEFIBIOS, not UEFI

اخترت isolinux قصداً — يشتغل على أي جهاز x86 تقريباً يدعم BIOS/Legacy، من أجهزة قديمة جداً لين أجهزة اليوم بوضع التوافق. isolinux was a deliberate choice — it boots on nearly any x86 machine with legacy BIOS support, from very old hardware to today's machines in compatibility mode.

04

مفتوح تسويه انتهBuilt to be forked

مب نظام نهائي مقفل. init script واحد بسيط تقدر تفتحه وتغيره، وتبني نسختك الخاصة فوگه. This isn't a closed, finished product. It's one simple init script you can open, edit, and build your own version on top of.

02 / طريقة العملHOW IT WORKS

من ضغطة الزر لين الشلFrom power button to shell

سلسلة إقلاع حقيقية، خطوة بخطوة — نفس السلسلة اللي تصير كل مرة تشغل فيها الجهاز. A real boot chain, step by step — the same sequence that runs every single time the machine powers on.

1

BIOS / SeaBIOS

الفيرموير يفحص الأجهزة ويسلّم التحكم لأول قطاع إقلاع يلقاه بالـCD/USB. Firmware initializes hardware and hands control to the first bootable sector it finds on the CD/USB.

2

ISOLINUX bootloader

يقرأ isolinux.cfg، يحمّل الكيرنل والـ initrd للذاكرة، ويقفز لهم. Reads isolinux.cfg, loads the kernel and initrd into memory, and jumps to them.

3

vmlinuz — Linux 6.6.0

الكيرنل يشتغل، يهيئ الذاكرة والمعالج، ويفك ضغط initrd.gz كنظام ملفات مؤقت بالرام. The kernel boots, initializes memory and CPU, and unpacks initrd.gz as a temporary in-RAM filesystem.

4

/init — busybox sh

سكربت شل واحد يركب /proc /sys /dev، ويحمّل موديولات USB/HID للماوس ولوحة المفاتيح. One shell script mounts /proc /sys /dev, and loads USB/HID kernel modules for mouse and keyboard.

5

القرار الأخيرThe final fork

Micro يوديك على شل نصي مباشرة. GUI يشغل xinit → Xorg → dwm + xterm فوگ نفس الـ init بالضبط. Micro drops you straight into a text shell. GUI runs xinit → Xorg → dwm + xterm — on the exact same init.
